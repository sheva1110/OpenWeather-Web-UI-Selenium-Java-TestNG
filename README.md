## Installation
```
Install JDK 11.x
Install Maven
---

### Execution

```
> Execute the command to run all tests in the project

```
	$ mvn clean test
```
> Execute the command to run only one test class in the project

```
	$ mvn clean test -Dtest=SearchWeatherTest
```
